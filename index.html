Internship

Everlabs

Year: 2024

Frontend developer

During my internship at Everlabs as a frontend developer, I worked on the Car Racing Typing Simulator project. Together with the team, we created a web application where users can compete in speed and accuracy of typing. The game has two main game modes: single and multiplayer.

Main functionality:

  • Single player mode : Players can train by typing and track their progress.
  • Multiplayer : Players can create or join rooms, compete against each other in real time.
    • Leaderboard : Leaderboard displays the best player scores.
  • Account Management : Users can create accounts, log in via email or Google.
  • Room Management : Users can create and customize rooms for multiplayer competitions.

The project was developed using React for the frontend part and Ruby on Rails for the backend. We used Tailwind CSS to style the interface, which allowed us to quickly and efficiently create an adaptive and attractive design.

My contribution to the project included the development of the frontend part of the following functionality:

  • Leaderboard : Create a leaderboard interface where the best player scores are displayed.
  • Multiplayer Room : Developing an interface for creating and managing rooms for multiplayer competitions.
  • Account : Implementation of the account management interface, including authorization via email and Google.

This experience has allowed me to significantly improve my skills in frontend application development and teamwork.
